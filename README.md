# ASTRA-B
TUA HACKATHON ASTRA-B Güneş Rüzgarı Erken Uyarı Ağı ASTRA-B Takımı
HELIOS, Güneş’ten gelen manyetik ve plazma parametre verilerini gerçek zamanlı olarak takip eden; bu verileri bir e-posta uyarı mekanizması ve arayüze dönüştüren kapsamlı bir sistemdir. Sistem, NOAA üzerinden sağlanan verileri simüle ederek işler ve sistemin aktifleştiği andan itibaren tüm verileri geriye dönük analizler için veri tabanında muhafaza eder.
